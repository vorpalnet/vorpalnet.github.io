Vorpal:BLADE Projects

Short for "Blended Layer Application Development Environment", BLADE is a collection of Java libraries and applications designed to simplify SIP Servlet development.

Service Javadocs Description
Admin Console org.vorpal.blade.applications.console Control the behavior of your applications through a web portal.
Framework org.vorpal.blade.framework A collection of Java libraries that simplify the creation of SIP Servlets beyond what's provided in JSR-359.
FSMAR org.vorpal.blade.library.fsmar2 Finite State Machine Application Router
Proxy-Balancer org.vorpal.blade.services.proxy.balancer A simple load balancer.
Proxy-Registrar org.vorpal.blade.proxy.registrar A small, elegant SIP proxy-registrar.
R3 Router org.vorpal.blade.services.proxy.router The Reductive Reasoning Router; Supports various search algorithms for optimal performance
ACL org.vorpal.blade.services.acl Access Control List; Allow or deny calls through the system.
CRUD org.vorpal.blade.services.crud Create, Read, Update & Delete; Transform any SIP message
Keep Alive org.vorpal.blade.services.keepalive Prevent sessions from timing out.
Options org.blade.services.options Control the behavior of options.
Presence org.blade.services.presence Maintains state of user endpoints
Queue org.blade.services.queue Maintains state of user endpoints
Transfer org.vorpal.blade.services.transfer Implements REFER for transfer applications
Test B2BUA org.vorpal.blade.test.b2bua An example B2BUA application
Test UAC org.vorpal.blade.test.uac A REST operated User Agent Client
Test UAS org.vorpal.blade.test.uas A test User Agent Server; Controllable through SIP Request-URI parameters
